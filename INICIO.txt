┌─ FLUJO DE USUARIO ────────────────────────────────────────────────────────┐
│                                                                           │
│ 👤 CREADOR DE CONTENIDO:                                                 │
│    1. Registrarse en /register                                          │
│    2. Iniciar sesión en /login                                          │
│    3. Ir a "Mi Página de Creador"                                       │
│    4. Completar setup (slug, bio, avatar)                              │
│    5. Agregar enlaces (Linktree)                                        │
│    6. Compartir URL pública: /@{slug}                                   │
│    7. Ver apoyos en dashboard                                           │
│                                                                           │
│ 👥 SEGUIDOR:                                                             │
│    1. Acceder a /@{slug}                                                │
│    2. Ver perfil, bio, enlaces                                         │
│    3. Click en "Apoyar al Creador"                                      │
│    4. Llenar formulario (nombre, mensaje, monto)                       │
│    5. Enviar apoyo                                                       │
│    6. ¡Éxito! Apoyo registrado                                          │
